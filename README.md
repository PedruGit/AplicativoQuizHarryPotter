# Quiz sobre Harry Potter

> Um aplicativo de perguntas e respostas sobre Harry Potter. Por decisão do professor, utilizamos o Google AI Studio para desenvolver o código. O objetivo da dinâmica foi provar que, mesmo com o auxílio de IA, é indispensável saber programar para construir um software funcional.

Este projeto foi desenvolvido para ser apresentado aos alunos da Escola Sandoval Soares de Azevedo na Feira Técnica de 2024.

---

## Instalação e Execução

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:

[Kotlin](https://developer.android.com/studio?hl=pt-br)

### Passo a Passo
**Clone o repositório:**

```bash
   git clone https://github.com/PedruGit/AplicativoQuizHarryPotter.git
```
